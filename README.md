# aorta-model-utilities

**NOTE:** This repository is *mirrored* to GitLab and the cardiac-biomechanics-lab group has access to this mirror. I primarily use GitHub for my own code development, so the mirroring is primarily a way to ensure that my code is accessible to anyone who needs utilities in the future.

This repository contains useful utilities for setting up FSI TBAD models. The primary focus for now will be on managing 4D Flow data. This includes segmentation and filtering the raw data.
